# waclaw-go — Daemon Go substituindo waclaw Node + wacli CLI — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Substituir o pipeline atual `wacli CLI binário + waclaw Node (child_process orchestrator)` por um único daemon Go (`waclaw-go`) que embeda `go.mau.fi/whatsmeow` diretamente, mantém o store SQLite com FTS5 em processo, expõe HTTP paridade com o waclaw Node atual, adiciona SSE push em `GET /events` consumido pelo `packages/daemon` TS já existente, e elimina toda a orquestração child_process/stdout/LOCK-file.

**Architecture:** Um processo Go, single-writer no store whatsmeow, N sessões como goroutines com handlers `whatsmeow.Client.AddEventHandler`. Event bus in-memory publica para persistência SQLite, download de mídia, e SSE broadcaster. HTTP server (`go-chi/chi`) monta endpoints de paridade mais `/events`. Daemon TS (`packages/daemon`) permanece inalterado — ele já espera SSE em `GET /events` com hipótese documentada; waclaw-go confirma a hipótese. Frontend PWA permanece inalterado — o proxy Next.js `/api/waclaw/[...path]` aponta para o mesmo host:porta.

**Tech Stack:** Go 1.23+, `go.mau.fi/whatsmeow` (mesma versão pinada que o upstream steipete/wacli usa hoje: `v0.0.0-20260211193157-7b33f6289f98`), `github.com/mattn/go-sqlite3` com build tag `sqlite_fts5`, `github.com/go-chi/chi/v5`, `github.com/rs/zerolog`. Build + deploy em `/home/orcabot/src/waclaw-go` no worker5 Tailscale `100.66.83.22`, systemd service `waclaw-go.service` user `orcabot`, porta `3100` (mesma do waclaw atual, via cutover controlado).

**⚠️ Dependências:** Este plano assume que:
1. O plano `2026-04-11-admin-multitenant.md` já foi mergeado — verificado com `git log --oneline main -20 | grep -i admin-multitenant` retornando commits como `f20d365 feat(admin): users table`.
2. O repo atual tem `packages/daemon/` com `waclaw-client.ts`, `forwarder.ts`, `index.ts`, e `packages/shared/src/validators/events.ts` definindo `OnAudioEventSchema`.
3. Há apenas **1 número WhatsApp** pareado em produção hoje (a conta "admin" em `/home/orcabot/.wacli/`). Re-pareamento desse número **uma vez** é aceitável como custo de migração, confirmado pelo usuário em 2026-04-11.

**⚠️ Não-dependência:** Este plano NÃO depende do plano `2026-04-11-wacli-emit-events.md`. Se aquele foi executado antes deste, as mudanças são descartadas na cutover (waclaw Node inteiro é removido). Se não foi executado, nada a fazer — waclaw-go substitui o caminho inteiro.

### Hipóteses que a Opção A adaptada mantinha que waclaw-go elimina

- ❌ `setInterval(wacli sync, 2min)` — não existe mais, waclaw Node já usa `spawn --follow`; waclaw-go elimina o spawn inteiro
- ❌ `wacli --emit-events` NDJSON stdout — não vamos patchar wacli; whatsmeow handler é in-process
- ❌ Child process LOCK file dance, `pauseLiveSync`, `runExclusive` queue — irrelevante em processo único que é dono do whatsmeow store
- ❌ Node readline parser de NDJSON — o bus é in-memory Go, publisher/subscriber pattern
- ❌ 700 LOC de `sessions.js` workarounds — deletado na cutover

### Trade-offs explícitos que aceito ao escolher este caminho

1. **~2500-3000 LOC Go novas** para escrever e manter. Justificativa: memória `feedback_definitive_architecture.md` registra preferência por arquitetura definitiva sobre MVP-mínimo; aderência mesmo com overhead upfront.
2. **Re-pareamento do único número produção**. 1 QR code, 1 vez. Custo zero dado o estado atual.
3. **Perda do CLI `wacli` em produção**. O binário continua instalado como dev tool para debug em stores não-produção (ex.: `wacli doctor /tmp/experiment-store`), mas não está no caminho do runtime. As features úteis (messages search, history backfill, doctor) são expostas via HTTP no waclaw-go.
4. **Manutenção de bump `go.mau.fi/whatsmeow`** a cada ~2-8 semanas quando a Meta atualiza o protocolo. Isso já era latente (via wacli upstream); agora é explícito e seu.
5. **Port-over do scheduler** (`waclaw_scheduled_messages` polling em `src/scheduler.js`). Este plano porta o scheduler para waclaw-go como Phase 11 — não como feature opcional.

### Responsabilidades por package

- **`cmd/waclaw-go/main.go`**: parse config, open store, build session manager, build event bus, build http server, wire dependencies, wait for SIGTERM/SIGINT, graceful shutdown em ordem (stop HTTP accept → stop scheduler → disconnect sessions → close store).

- **`internal/config`**: `type Config struct { APIKey string; Port int; StoreRootDir string; ... }`, função `FromEnv() (Config, error)` lê `WACLAW_API_KEY`, `PORT`, `SESSIONS_DIR`, `LOG_LEVEL`, `SUPABASE_URL`, `SUPABASE_SERVICE_ROLE_KEY` (pro scheduler), etc. Defaults documentados.

- **`internal/store`**: dono do SQLite. Cada `Session` tem sua própria instância de `*Store` apontando para o próprio diretório. O schema é embutido via `//go:embed migrations/0001_initial.sql`. Métodos retornam tipos do próprio package (`store.Chat`, `store.Message`) sem expor `sql.Row`. Tests usam `:memory:` SQLite (com build tag `sqlite_fts5` ativado no Makefile e no go test).

- **`internal/session`**: `Manager` gerencia um `map[sessionID]*Session`. `Session` envelopa `*whatsmeow.Client` + `*store.Store` + um `chan mediaJob` + um `context.Context` pra cancelamento. `handler.go` registra o event callback que dispatches:
  - `*events.Message` → `store.InsertMessage`, enfileira download, publica no `events.Bus`
  - `*events.HistorySync` → `store.InsertMessage` em batch
  - `*events.PushName` → `store.UpsertContact`
  - `*events.GroupInfo` → `store.UpsertGroup`
  - `*events.Connected`, `*events.Disconnected`, `*events.LoggedOut` → state transitions + bus publish
  - `*events.PairSuccess` → confirma pareamento
  - `*events.QR` → publica no QR channel per-session
  
  `media_queue.go` implementa um fan-out pattern: pool de N workers (default 4) consume `chan mediaJob`, chama `whatsmeow.Client.Download(msg)`, escreve arquivo no disco `{storeDir}/media/{chatJid}/{msgId}.{ext}`, atualiza `store.UpdateLocalPath`.

- **`internal/events`**: `Bus` é um pub/sub in-memory simples. `Publish(ev Event)` envia para cada subscriber via buffered channel. `Subscribe() (<-chan Event, func())` retorna um receiver + cleanup fn. Subscribers lentos sofrem drop (log warn). `translator.go` converte `wa.ParsedMessage` em `wire.MessageEvent` no formato que `packages/shared/src/validators/events.ts` `OnAudioEventSchema` espera.

- **`internal/httpserver`**: `chi` router com middleware stack `[Recoverer, RequestID, RealIP, zerolog-req-logger, AuthAPIKey]` exceto `/health` e `/events` (events tem auth própria via header). Um handler por endpoint em arquivos separados. Server tem reference ao `session.Manager`, `store` (via manager), `events.Bus`.

- **`internal/avatars`**: cache de profile pictures em `{storeDir}/avatars/{sanitizedJid}.jpg`. Download via `whatsmeow.Client.GetProfilePictureInfo` + HTTP GET da URL retornada.

- **`internal/scheduler`**: porta de `waclaw/src/scheduler.js`. Loop com `time.Ticker` 30s, query Supabase `waclaw_scheduled_messages WHERE status='pending' AND scheduled_for <= NOW()`, claim row, chama `session.SendText`, update row `status='sent'|'failed'`.

## Rollback plan (leia antes de começar qualquer fase ≥ 10)

Cutover para waclaw-go envolve parar `waclaw.service` Node e iniciar `waclaw-go.service`. Se falhar:

```bash
ssh root@100.66.83.22

# 1) Parar waclaw-go
systemctl stop waclaw-go.service

# 2) Subir waclaw Node de volta
systemctl start waclaw.service
systemctl is-active waclaw.service   # expected: active

# 3) Se waclaw Node também falhar (unlikely pois é o que estava rodando):
#    restaurar do backup
rm -rf /opt/waclaw
cp -a /opt/waclaw.bak-<timestamp> /opt/waclaw
systemctl start waclaw.service

# 4) Reparear se store whatsmeow ficou em estado inconsistente:
#    parar, limpar LOCK, rearmar auth
systemctl stop waclaw.service
rm -f /home/orcabot/.wacli/LOCK
# (auth via frontend PWA → Settings → reparear instance)
systemctl start waclaw.service

# 5) Conferir
curl -s -H "X-API-Key: waclaw-dev-key" http://localhost:3100/health
journalctl -u waclaw.service -n 30 --no-pager
```

```go
// Package config centralizes environment-based configuration.
// All env vars and their defaults live here so main.go stays small.
package config

import (
	"errors"
	"fmt"
	"os"
	"strconv"
)

// Config holds runtime configuration. Immutable after FromEnv() returns.
type Config struct {
	// HTTP
	APIKey   string
	Port     int
	BindHost string

	// Storage
	SessionsDir string

	// Logging
	LogLevel  string
	LogFormat string

	// Supabase (scheduler)
	SupabaseURL        string
	SupabaseServiceKey string

	// Media
	MediaDownloadWorkers int
	MaxAudioBytes        int64
}

const (
	defaultPort             = 3100
	defaultBindHost         = "0.0.0.0"
	defaultSessionsDir      = "/home/orcabot/waclaw-go/sessions"
	defaultLogLevel         = "info"
	defaultLogFormat        = "json"
	defaultMediaWorkers     = 4
	defaultMaxAudioBytes    = 25 * 1024 * 1024 // 25 MiB, matches zapi-shared
)

// FromEnv reads all configuration from environment variables.
// Returns an error if any required value is missing or malformed.
func FromEnv() (Config, error) {
	cfg := Config{
		APIKey:               os.Getenv("WACLAW_API_KEY"),
		BindHost:             getEnvDefault("BIND_HOST", defaultBindHost),
		SessionsDir:          getEnvDefault("SESSIONS_DIR", defaultSessionsDir),
		LogLevel:             getEnvDefault("LOG_LEVEL", defaultLogLevel),
		LogFormat:            getEnvDefault("LOG_FORMAT", defaultLogFormat),
		SupabaseURL:          os.Getenv("SUPABASE_URL"),
		SupabaseServiceKey:   os.Getenv("SUPABASE_SERVICE_ROLE_KEY"),
	}

	if cfg.APIKey == "" {
		return Config{}, errors.New("WACLAW_API_KEY must be set")
	}

	port, err := parseIntDefault("PORT", defaultPort)
	if err != nil {
		return Config{}, err
	}
	cfg.Port = port

	workers, err := parseIntDefault("MEDIA_DOWNLOAD_WORKERS", defaultMediaWorkers)
	if err != nil {
		return Config{}, err
	}
	if workers < 1 {
		workers = 1
	}
	cfg.MediaDownloadWorkers = workers

	maxAudio, err := parseInt64Default("MAX_AUDIO_BYTES", defaultMaxAudioBytes)
	if err != nil {
		return Config{}, err
	}
	cfg.MaxAudioBytes = maxAudio

	return cfg, nil
}

func getEnvDefault(key, def string) string {
	if v := os.Getenv(key); v != "" {
		return v
	}
	return def
}

func parseIntDefault(key string, def int) (int, error) {
	raw := os.Getenv(key)
	if raw == "" {
		return def, nil
	}
	v, err := strconv.Atoi(raw)
	if err != nil {
		return 0, fmt.Errorf("env %s: %w", key, err)
	}
	return v, nil
}

func parseInt64Default(key string, def int64) (int64, error) {
	raw := os.Getenv(key)
	if raw == "" {
		return def, nil
	}
	v, err := strconv.ParseInt(raw, 10, 64)
	if err != nil {
		return 0, fmt.Errorf("env %s: %w", key, err)
	}
	return v, nil
}
```

```go
// Store wraps a single SQLite database connection. Methods on Store are
// safe for concurrent use — sql.DB is safe for concurrent use by default
// and mattn/go-sqlite3 serializes writes internally.
type Store struct {
	db  *sql.DB
	dsn string
}

// Open creates or opens a SQLite database at path, enables WAL and
// foreign keys, and applies the app-level migrations.
// The whatsmeow_* tables are created later, when a whatsmeow container
// is opened against the same file.
func Open(path string) (*Store, error) {
	if path == "" {
		return nil, errors.New("store path is empty")
	}
	dsn := fmt.Sprintf("file:%s?_journal=WAL&_busy_timeout=10000&_foreign_keys=on", path)
	db, err := sql.Open("sqlite3", dsn)
	if err != nil {
		return nil, fmt.Errorf("sql.Open: %w", err)
	}
	// Keep a small pool — SQLite is single-writer.
	db.SetMaxOpenConns(4)
	db.SetMaxIdleConns(2)

	s := &Store{db: db, dsn: dsn}
	if err := s.migrate(); err != nil {
		_ = db.Close()
		return nil, fmt.Errorf("migrate: %w", err)
	}
	return s, nil
}

// Close releases the underlying database handle.
func (s *Store) Close() error {
	return s.db.Close()
}

// DB exposes the raw handle for packages that need to hand it to
// whatsmeow/store/sqlstore. External callers should prefer the typed
// methods on Store.
func (s *Store) DB() *sql.DB {
	return s.db
}

// DSN returns the connection string used to open this store. Used by the
// session package to open a whatsmeow sqlstore container on the same file.
func (s *Store) DSN() string {
	return s.dsn
}

func (s *Store) migrate() error {
	if _, err := s.db.Exec(migration0001); err != nil {
		return err
	}
	// Record that 0001 was applied. Idempotent via INSERT OR IGNORE.
	_, err := s.db.Exec(
		`INSERT OR IGNORE INTO schema_migrations (version, name, applied_at) VALUES (?, ?, strftime('%s','now'))`,
		1, "0001_initial",
	)
	return err
}
```

```go
package store

import "database/sql"

// Chat is a row in the chats table. Name comes from the first non-empty
// of group name, contact full_name/business_name/push_name, or the chats.name.
// GetChats resolves the join at query time; callers of UpsertChat just pass
// whatever they have — the display resolution is for reads, not writes.
type Chat struct {
	JID           string
	Kind          string // dm | group | broadcast | unknown
	Name          string
	LastMessageTs int64

	// Read-only fields populated by GetChats:
	LastMessage string
	LastSender  string
	MsgCount    int64
	IsGroup     bool
}

// UpsertChat inserts or updates a single chat row keyed by JID.
// LastMessageTs is only updated if the new value is greater than the
// existing one, so a race between InsertMessage and UpsertChat cannot
// regress the timestamp.
func (s *Store) UpsertChat(c Chat) error {
	_, err := s.db.Exec(`
		INSERT INTO chats (jid, kind, name, last_message_ts)
		VALUES (?, ?, ?, ?)
		ON CONFLICT(jid) DO UPDATE SET
			kind = excluded.kind,
			name = COALESCE(NULLIF(excluded.name, ''), chats.name),
			last_message_ts = CASE
				WHEN excluded.last_message_ts > chats.last_message_ts THEN excluded.last_message_ts
				ELSE chats.last_message_ts
			END
	`, c.JID, c.Kind, c.Name, c.LastMessageTs)
	return err
}

// GetChats returns active chats (last_message_ts > 0) ordered by
// last message time descending. The query mirrors waclaw Node's
// src/db.js getChats() exactly, including the COALESCE resolution of
// display name across groups, contacts, and chat name.
func (s *Store) GetChats() ([]Chat, error) {
	rows, err := s.db.Query(`
		SELECT 
			c.jid,
			COALESCE(
				NULLIF(g.name, ''),
				NULLIF(ct.full_name, ''),
				NULLIF(ct.business_name, ''),
				NULLIF(ct.push_name, ''),
				NULLIF(c.name, ''),
				c.jid
			) AS name,
			c.kind,
			c.last_message_ts,
			(SELECT COUNT(*) FROM messages m WHERE m.chat_jid = c.jid) AS msg_count,
			(SELECT text FROM messages m WHERE m.chat_jid = c.jid ORDER BY ts DESC LIMIT 1) AS last_message,
			(SELECT sender_name FROM messages m WHERE m.chat_jid = c.jid ORDER BY ts DESC LIMIT 1) AS last_sender
		FROM chats c
		LEFT JOIN contacts ct ON c.jid = ct.jid
		LEFT JOIN groups g ON c.jid = g.jid
		WHERE c.last_message_ts > 0
		ORDER BY c.last_message_ts DESC
	`)
	if err != nil {
		return nil, err
	}
	defer rows.Close()

	var out []Chat
	for rows.Next() {
		var c Chat
		var lastMsg, lastSender sql.NullString
		if err := rows.Scan(&c.JID, &c.Name, &c.Kind, &c.LastMessageTs, &c.MsgCount, &lastMsg, &lastSender); err != nil {
			return nil, err
		}
		c.LastMessage = lastMsg.String
		c.LastSender = lastSender.String
		c.IsGroup = c.Kind == "group"
		out = append(out, c)
	}
	return out, rows.Err()
}
```

```go
package store

import "database/sql"

// Message is a row in the messages table. Byte slices (MediaKey, FileSHA256,
// FileEncSHA256) are stored BLOB and carry the same bytes whatsmeow gave us.
type Message struct {
	Rowid         int64
	ChatJID       string
	ChatName      string
	MsgID         string
	SenderJID     string
	SenderName    string
	Ts            int64
	FromMe        bool
	Text          string
	DisplayText   string
	MediaType     string
	MediaCaption  string
	Filename      string
	MimeType      string
	DirectPath    string
	MediaKey      []byte
	FileSHA256    []byte
	FileEncSHA256 []byte
	FileLength    int64
	LocalPath     string
	DownloadedAt  int64
}

// InsertMessage inserts a new message row. Uses INSERT OR IGNORE on the
// unique (chat_jid, msg_id) constraint so duplicate delivery (HistorySync
// replay after reconnect) is silent and cheap. The FTS5 triggers fire
// automatically via the schema.
func (s *Store) InsertMessage(m Message) error {
	_, err := s.db.Exec(`
		INSERT OR IGNORE INTO messages (
			chat_jid, chat_name, msg_id, sender_jid, sender_name, ts, from_me,
			text, display_text, media_type, media_caption, filename, mime_type,
			direct_path, media_key, file_sha256, file_enc_sha256, file_length,
			local_path, downloaded_at
		) VALUES (?,?,?,?,?,?,?,?,?,?,?,?,?,?,?,?,?,?,?,?)
	`,
		m.ChatJID, nullable(m.ChatName), m.MsgID, nullable(m.SenderJID), nullable(m.SenderName),
		m.Ts, boolToInt(m.FromMe),
		nullable(m.Text), nullable(m.DisplayText),
		nullable(m.MediaType), nullable(m.MediaCaption), nullable(m.Filename), nullable(m.MimeType),
		nullable(m.DirectPath), m.MediaKey, m.FileSHA256, m.FileEncSHA256,
		nullableInt(m.FileLength), nullable(m.LocalPath), nullableInt(m.DownloadedAt),
	)
	return err
}

// GetMessagesByChat returns up to limit messages for a chat, ordered
// chronologically ascending (oldest first) — matching the waclaw Node
// behavior which SELECTs ORDER BY ts DESC then reverses the slice.
//
// If beforeTs > 0, only returns messages with ts < beforeTs (pagination cursor).
func (s *Store) GetMessagesByChat(chatJID string, limit int, beforeTs int64) ([]Message, error) {
	var rows *sql.Rows
	var err error
	if beforeTs > 0 {
		rows, err = s.db.Query(`
			SELECT `+messageCols+` FROM messages
			WHERE chat_jid = ? AND ts < ?
			ORDER BY ts DESC LIMIT ?`,
			chatJID, beforeTs, limit)
	} else {
		rows, err = s.db.Query(`
			SELECT `+messageCols+` FROM messages
			WHERE chat_jid = ?
			ORDER BY ts DESC LIMIT ?`,
			chatJID, limit)
	}
	if err != nil {
		return nil, err
	}
	defer rows.Close()

	var collected []Message
	for rows.Next() {
		m, err := scanMessage(rows)
		if err != nil {
			return nil, err
		}
		collected = append(collected, m)
	}
	if err := rows.Err(); err != nil {
		return nil, err
	}

	// Reverse to chronological ASC.
	for i, j := 0, len(collected)-1; i < j; i, j = i+1, j-1 {
		collected[i], collected[j] = collected[j], collected[i]
	}
	return collected, nil
}

// GetMessageByID returns a single message or (nil, nil) if not found.
func (s *Store) GetMessageByID(chatJID, msgID string) (*Message, error) {
	row := s.db.QueryRow(`SELECT `+messageCols+` FROM messages WHERE chat_jid = ? AND msg_id = ?`,

// SearchMessages runs an FTS5 MATCH query across all indexed columns.
// Results are ordered by ts DESC, up to limit rows.
func (s *Store) SearchMessages(query string, limit int) ([]Message, error) {
	rows, err := s.db.Query(`
		SELECT `+prefixedMessageCols("m.")+` FROM messages_fts fts
		JOIN messages m ON m.rowid = fts.rowid
		WHERE messages_fts MATCH ?
		ORDER BY m.ts DESC LIMIT ?`,
		query, limit)
	if err != nil {
		return nil, err
	}
	defer rows.Close()

	var out []Message
	for rows.Next() {
		m, err := scanMessage(rows)
		if err != nil {
			return nil, err
		}
		out = append(out, m)
	}
	return out, rows.Err()
}

		WHERE chat_jid = ? AND msg_id = ?`,

	return err
```

```go
// NewManager reads existing session directories from sessionsRoot and

func NewManager(sessionsRoot string, log zerolog.Logger, handlerDeps any) (*Manager, error) {

	return m, nil

// loadExisting scans sessionsRoot for existing session directories.
// Each directory name is treated as a session ID; the Store is opened
// lazily on demand (Task 3.4 connects).
func (m *Manager) loadExisting() error {
	entries, err := os.ReadDir(m.sessionsRoot)
	if err != nil {
		return err
	}
	for _, e := range entries {
		if !e.IsDir() {
			continue
		}
		id := e.Name()
		dir := filepath.Join(m.sessionsRoot, id)
		sess := &Session{
			ID:          id,
			StoreDir:    dir,
			log:         m.log.With().Str("session", id[:8]).Logger(),
			state:       StateNew,
			handlerDeps: m.handlerDeps,
		}

		m.log.Info().Str("session", id[:8]).Msg("loaded existing session dir")

	return nil

		return nil, fmt.Errorf("mkdir session dir: %w", err)
```

```go
// If the device has no JID yet (never paired), Connect will request a QR
// code through client.GetQRChannel. The latest QR string is stored in
// s.lastQR and exposed via LastQR(); callers poll that field (HTTP GET
// /qr) until the user scans.
//
// Idempotent: if already connected, returns nil.
func (s *Session) Connect(ctx context.Context) error {
	s.mu.Lock()
	switch s.state {
	case StateConnected, StateConnecting:
		s.mu.Unlock()
		return nil
	case StateClosed:
		s.mu.Unlock()
		return errClosed
	}
	s.state = StateConnecting
	s.mu.Unlock()

	// Open or re-use the app-level store.
	if s.store == nil {
		st, err := store.Open(filepath.Join(s.StoreDir, "waclaw.db"))
		if err != nil {
			s.fail(fmt.Errorf("store open: %w", err))
			return err
		}
		s.store = st
	}

	// Open whatsmeow sqlstore container on the SAME file (it creates its
	// own whatsmeow_* tables alongside our app-level tables).
	waLogger := WACliLogAdapter{zl: s.log}
	container, err := waStoreSQL.New("sqlite3", s.store.DSN(), waLogger.Sub("sqlstore"))
	if err != nil {
		s.fail(fmt.Errorf("whatsmeow sqlstore.New: %w", err))
		return err
	}
	s.container = container

	// Load first device or create new.
	device, err := container.GetFirstDevice()
	if err != nil {
		s.fail(fmt.Errorf("GetFirstDevice: %w", err))
		return err
	}

	client := whatsmeow.NewClient(device, waLogger.Sub("client"))
	s.client = client
	client.AddEventHandler(s.buildEventHandler())

	if device.ID == nil {
		// Needs pairing.
		qrChan, err := client.GetQRChannel(ctx)
		if err != nil {
			s.fail(fmt.Errorf("GetQRChannel: %w", err))
			return err
		}
		if err := client.Connect(); err != nil {
			s.fail(fmt.Errorf("Connect (pre-QR): %w", err))
			return err
		}
		s.setState(StateWaitingQR)

		go s.watchQR(qrChan)
	} else {
		// Already paired, just connect.
		if err := client.Connect(); err != nil {
			s.fail(fmt.Errorf("Connect: %w", err))
			return err
		}
		// State transitions to StateConnected via the Connected event in
		// the handler (Task 4.5).
	}

	return nil
}

// watchQR drains the QR channel, storing each new QR string until the
// user scans (PairSuccess event in the handler transitions state). Timeout
// comes from the channel being closed by whatsmeow.
func (s *Session) watchQR(qrChan <-chan whatsmeow.QRChannelItem) {
	for item := range qrChan {
		switch item.Event {
		case "code":
			s.log.Info().Msg("new QR code")
			s.setLastQR(item.Code)
		case "success":
			s.log.Info().Msg("QR pairing success")
			s.setLastQR("")
			return
		case "timeout":
			s.log.Warn().Msg("QR pairing timed out")
			s.fail(errors.New("QR pairing timed out"))
			return
		default:
			s.log.Debug().Str("qr_event", item.Event).Msg("QR channel event")
		}
	}
}

func (s *Session) fail(err error) {
	s.mu.Lock()
	s.state = StateDisconnected
	s.lastError = err.Error()
	s.mu.Unlock()
	s.log.Error().Err(err).Msg("session fail")
}

// WaitForConnected blocks until state becomes StateConnected or ctx expires.
// Used by HTTP /auth to return connection status synchronously.
func (s *Session) WaitForConnected(ctx context.Context, timeout time.Duration) error {
	deadline := time.NewTimer(timeout)
	defer deadline.Stop()
	tick := time.NewTicker(200 * time.Millisecond)
	defer tick.Stop()
	for {
		if s.State() == StateConnected {
			return nil
		}
		select {
		case <-ctx.Done():
			return ctx.Err()
		case <-deadline.C:
			return errors.New("timeout waiting for connected")
		case <-tick.C:
		}
	}
}
```

```go
// extractFromProto copies text and media fields from a whatsmeow proto
// message into store.Message. Handles the major message types we care
// about: text (conversation / extended), image, video, audio (PTT),
// document, sticker. Other types leave media_type empty.
func extractFromProto(pm *waProto.Message, m *store.Message) {
	if pm == nil {
		return
	}

	// Text — conversation (simple) or extended text (with formatting).
	if t := pm.GetConversation(); t != "" {
		m.Text = t
		m.DisplayText = t
	} else if ex := pm.GetExtendedTextMessage(); ex != nil {
		m.Text = ex.GetText()
		m.DisplayText = ex.GetText()
	}

	if img := pm.GetImageMessage(); img != nil {
		m.MediaType = "image"
		m.MimeType = img.GetMimetype()
		m.MediaCaption = img.GetCaption()
		m.DirectPath = img.GetDirectPath()
		m.MediaKey = img.GetMediaKey()
		m.FileSHA256 = img.GetFileSHA256()
		m.FileEncSHA256 = img.GetFileEncSHA256()
		m.FileLength = int64(img.GetFileLength())
		if m.DisplayText == "" {
			m.DisplayText = img.GetCaption()
		}
		return
	}

	// Video.
	if vid := pm.GetVideoMessage(); vid != nil {
		m.MediaType = "video"
		m.MimeType = vid.GetMimetype()
		m.MediaCaption = vid.GetCaption()
		m.DirectPath = vid.GetDirectPath()
		m.MediaKey = vid.GetMediaKey()
		m.FileSHA256 = vid.GetFileSHA256()
		m.FileEncSHA256 = vid.GetFileEncSHA256()
		m.FileLength = int64(vid.GetFileLength())
		if m.DisplayText == "" {
			m.DisplayText = vid.GetCaption()
		}
		return
	}

	if aud := pm.GetAudioMessage(); aud != nil {

		}

		m.FileSHA256 = aud.GetFileSHA256()
		m.FileEncSHA256 = aud.GetFileEncSHA256()
		m.FileLength = int64(aud.GetFileLength())
		return
	}

	// Document.
	if doc := pm.GetDocumentMessage(); doc != nil {
		m.MediaType = "document"
		m.MimeType = doc.GetMimetype()
		m.Filename = doc.GetFileName()
		m.MediaCaption = doc.GetCaption()
		m.DirectPath = doc.GetDirectPath()
		m.MediaKey = doc.GetMediaKey()
		m.FileSHA256 = doc.GetFileSHA256()

		m.FileSHA256 = sti.GetFileSHA256()
		m.FileEncSHA256 = sti.GetFileEncSHA256()
		m.FileLength = int64(sti.GetFileLength())
		return
	}

	if m.DisplayText == "" && m.Text == "" {

	}
}
```

```go
// MediaQueueRunner runs N worker goroutines that consume MediaJob jobs,
// call whatsmeow.Client.DownloadAny, write the bytes to disk, and update
// the store. Single instance shared across all sessions; workers look
// the session up by ID per job.
type MediaQueueRunner struct {
	jobs    <-chan MediaJob
	manager *Manager
	workers int
}

// NewMediaQueueRunner wires a runner. Caller creates the channel and
// passes it to both the Manager's HandlerDeps and the runner so the
// producer and consumer share the same chan.
func NewMediaQueueRunner(jobs <-chan MediaJob, mgr *Manager, workers int) *MediaQueueRunner {
	if workers < 1 {
		workers = 1
	}
	return &MediaQueueRunner{jobs: jobs, manager: mgr, workers: workers}
}

// Start launches worker goroutines. Returns when ctx is cancelled.
func (r *MediaQueueRunner) Start(ctx context.Context) {
	for i := 0; i < r.workers; i++ {
		go r.worker(ctx, i)
	}
}

func (r *MediaQueueRunner) worker(ctx context.Context, idx int) {
	for {
		select {
		case <-ctx.Done():
			return
		case job, ok := <-r.jobs:
			if !ok {
				return
			}
			r.process(ctx, job, idx)
		}
	}
}

func (r *MediaQueueRunner) process(ctx context.Context, job MediaJob, idx int) {
	sess, err := r.manager.Get(job.SessionID)
	if err != nil {
		return
	}

	m, err := sess.store.GetMessageByID(job.ChatJID, job.MsgID)
	if err != nil || m == nil {
		return
	}
	if m.LocalPath != "" {
		return // already downloaded
	}
	if m.MediaType == "" || m.DirectPath == "" || len(m.MediaKey) == 0 {
		return // nothing to download
	}

	// DownloadableMessage is an interface used by whatsmeow's Download.
	// We reconstruct one from stored fields. The concrete implementation
	// depends on the MediaType, so we build a minimal wrapper struct.
	target := filepath.Join(sess.StoreDir, "media", job.ChatJID, job.MsgID)
	if err := os.MkdirAll(filepath.Dir(target), 0o700); err != nil {
		sess.log.Warn().Err(err).Msg("mkdir media")
		return
	}

	data, err := sess.client.DownloadMediaWithPath(
		ctx,
		m.DirectPath,
		m.FileEncSHA256,
		m.FileSHA256,
		m.MediaKey,
		int(m.FileLength),
		mediaTypeFor(m.MediaType),
		whatsmeow.MediaTypeOfMMS(mediaTypeFor(m.MediaType)),
	)
	if err != nil {
		sess.log.Warn().Err(err).Str("msg_id", job.MsgID).Msg("download failed")
		return
	}

	if err := os.WriteFile(target, data, 0o600); err != nil {
		sess.log.Warn().Err(err).Msg("write media file")
		return
	}

	if err := sess.store.UpdateLocalPath(job.ChatJID, job.MsgID, target, time.Now().Unix()); err != nil {
		sess.log.Warn().Err(err).Msg("UpdateLocalPath")
	}
	sess.log.Debug().
		Int("worker", idx).
		Str("msg_id", job.MsgID).
		Int("bytes", len(data)).
		Msg("media downloaded")
}

// mediaTypeFor maps our store media_type values to whatsmeow's
// whatsmeow.MediaType constants used for download decryption.

		return whatsmeow.MediaDocument
	case "sticker":
		return whatsmeow.MediaImage // stickers use image type for decryption
	default:
		return whatsmeow.MediaImage
	}
}
```

```go
// Package events is the in-memory pub/sub bus and event wire types.
// All event producers (session.handleMessage) and consumers
// (httpserver.ServeEventsSSE) talk through a single *Bus.
package events

import (
	"encoding/json"
	"sync"
	"sync/atomic"
)

// Event is one NDJSON-like event emitted by the session handler and
// consumed by SSE subscribers.
// Type is the event name; Raw is a pre-marshaled JSON blob (already
// translated to the wire shape the daemon TS expects — see translator.go).
type Event struct {
	Type string
	Raw  json.RawMessage
}

// Bus is a concurrent in-memory pub/sub. Subscribers receive only events
// published after they subscribed; there is no replay/backlog.
//
// Publishers do not block on slow subscribers: the send uses a non-blocking
// select and drops if the subscriber's channel buffer is full. Drops are
// counted per-subscriber for observability.
type Bus struct {
	mu        sync.RWMutex
	nextID    uint64
	subs      map[uint64]*subscriber
}

type subscriber struct {
	ch     chan Event
	closed atomic.Bool
	drops  atomic.Uint64
}

// NewBus constructs an empty bus.
func NewBus() *Bus {
	return &Bus{subs: make(map[uint64]*subscriber)}
}

// Subscribe returns a receive-only channel and an unsubscribe function.
// bufSize is the per-subscriber channel buffer; a reasonable default is 32.
// Callers must drain ch or accept drops.
//
// Calling unsub more than once is safe.
func (b *Bus) Subscribe(bufSize int) (<-chan Event, func()) {
	if bufSize < 1 {
		bufSize = 1
	}
	sub := &subscriber{ch: make(chan Event, bufSize)}

	b.mu.Lock()
	b.nextID++
	id := b.nextID
	b.subs[id] = sub
	b.mu.Unlock()

	unsub := func() {
		b.mu.Lock()
		if _, ok := b.subs[id]; ok {
			delete(b.subs, id)
		}
		b.mu.Unlock()
		if sub.closed.CompareAndSwap(false, true) {
			close(sub.ch)
		}
	}
	return sub.ch, unsub
}

// Publish delivers evt to every current subscriber. Slow subscribers
// that cannot accept the send immediately have it dropped (metric in
// subscriber.drops) so one stuck consumer cannot stall producers.
func (b *Bus) Publish(evt Event) {
	b.mu.RLock()
	defer b.mu.RUnlock()

	for _, sub := range b.subs {
		if sub.closed.Load() {
			continue
		}
		select {
		case sub.ch <- evt:
		default:
			sub.drops.Add(1)
		}
	}
}

// SubscriberCount returns the current live subscriber count.
// Used by /health for observability.
func (b *Bus) SubscriberCount() int {
	b.mu.RLock()
	defer b.mu.RUnlock()
	return len(b.subs)
}
```

```go
// after translation. Sent as the outer envelope's `message` field, with
// outer `type: "message"` and outer `session_id`. The shape mirrors what
// packages/daemon/src/waclaw-client.ts's extractAudioEvent() reads and

// on the Next.js side — kept in sync manually since Go can't import Zod.

// serialization in translator.go.
type WireMessageEvent struct {
	ID         string     `json:"id"`
	ChatJID    string     `json:"chat_jid"`
	ChatName   string     `json:"chat_name"`
	From       string     `json:"from"`         // sender phone / JID user part
	SenderName string     `json:"sender_name"`
	FromMe     bool       `json:"from_me"`
	Timestamp  int64      `json:"timestamp"`    // unix seconds
	Text       string     `json:"text,omitempty"`
	Audio      *WireAudio `json:"audio,omitempty"`
}

// URL is a HTTP(S) URL to a waclaw-go /sessions/:id/media/:jid/:msgId

	DurationSeconds int    `json:"duration_seconds"`
	SizeBytes       int64  `json:"size_bytes"`

// data: {"type":"message","session_id":"...","message":{...}}\n
```

```go
// TranslateInput holds the minimum fields the translator needs to
// produce a WireEnvelope. Composed in session.handleMessage from the
// store.Message we just persisted plus the session id and the base URL
// for media links. Kept deliberately flat — no dep on internal/store or
// internal/session to avoid import cycles.
type TranslateInput struct {
	SessionID      string
	MessageID      string
	ChatJID        string
	ChatName       string
	SenderJID      string
	SenderName     string
	FromMe         bool
	Timestamp      int64
	Text           string
	MediaType      string // text | image | audio | ptt | video | document | sticker
	MimeType       string
	FileLength     int64
	DurationSeconds int

	// MediaBaseURL is the base URL served by waclaw-go for streaming
	// media, e.g. "http://localhost:3100". The translator appends
	// "/sessions/:id/media/:chatJid/:msgId".
	MediaBaseURL string
}

// TranslateMessage builds a WireEnvelope ready to Publish on the bus.
// Returns an Event with Type set and Raw pre-marshaled.
func TranslateMessage(in TranslateInput) (Event, error) {
	msg := &WireMessageEvent{
		ID:         in.MessageID,
		ChatJID:    in.ChatJID,
		ChatName:   in.ChatName,
		From:       jidUserPart(in.SenderJID),
		SenderName: in.SenderName,
		FromMe:     in.FromMe,
		Timestamp:  in.Timestamp,
		Text:       in.Text,
	}

	if in.MediaType == "audio" || in.MediaType == "ptt" {
		msg.Audio = &WireAudio{
			URL: fmt.Sprintf("%s/sessions/%s/media/%s/%s",
				strings.TrimRight(in.MediaBaseURL, "/"),
				in.SessionID,
				in.ChatJID,
				in.MessageID,
			),
			DurationSeconds: in.DurationSeconds,
			SizeBytes:       in.FileLength,
		}
	}

	env := WireEnvelope{
		Type:      "message",
		SessionID: in.SessionID,
		Message:   msg,
	}
	raw, err := json.Marshal(env)
	if err != nil {
		return Event{}, fmt.Errorf("marshal envelope: %w", err)
	}
	return Event{Type: "message", Raw: raw}, nil
}

// jidUserPart returns the "5511987654321" part of "5511987654321@s.whatsapp.net".
// Returns empty for group JIDs (ending in @g.us) since there's no sender phone
// at the group level — callers should use SenderJID's user part instead.
func jidUserPart(jid string) string {
	at := strings.IndexByte(jid, '@')
	if at <= 0 {
		return jid
	}
	return jid[:at]
}
```

```go
	Bus     *events.Bus
	Log     zerolog.Logger
	APIKey  string
}

// Server wraps http.Server plus chi router. New returns a non-started
// server; callers call Run(ctx) to block until shutdown.
type Server struct {
	http   *http.Server
	router *chi.Mux
	deps   Deps
}

// New wires the router, middleware, and handlers. Auth middleware is
// applied globally except for /health and /events (events has its own
// auth because SSE can't set custom headers from EventSource).
func New(addr string, deps Deps) *Server {
	r := chi.NewRouter()
	r.Use(chimw.RequestID)
	r.Use(chimw.RealIP)
	r.Use(chimw.Recoverer)
	r.Use(RequestLogger(deps.Log))

	srv := &Server{
		router: r,
		deps:   deps,
		http: &http.Server{
			Addr:              addr,
			Handler:           r,
			ReadHeaderTimeout: 10 * time.Second,
			// No global write timeout so SSE streams can stay open.
		},
	}

	srv.mountRoutes()
	return srv
}

// Run blocks serving until ctx is cancelled, then gracefully shuts down.
func (s *Server) Run(ctx context.Context) error {
	errCh := make(chan error, 1)
	go func() {
		s.deps.Log.Info().Str("addr", s.http.Addr).Msg("http listening")
		err := s.http.ListenAndServe()
		if err != nil && !errors.Is(err, http.ErrServerClosed) {
			errCh <- err
		}
		close(errCh)
	}()

	select {
	case err := <-errCh:
		return err
	case <-ctx.Done():
		s.deps.Log.Info().Msg("http shutting down")
		shutdownCtx, cancel := context.WithTimeout(context.Background(), 5*time.Second)
		defer cancel()
		return s.http.Shutdown(shutdownCtx)
	}
}

// mountRoutes attaches handlers to the router. Each category lives in
// its own file (sessions.go, chats.go, etc.). Auth middleware is applied
// to all routes inside the protected group.
func (s *Server) mountRoutes() {

	// Public — no auth.
	r.Get("/health", s.handleHealth)

	// its own auth inline — see events_sse.go).

// Content-Type and encodes the body. Errors are logged but not

func writeJSON(w http.ResponseWriter, status int, body any) {

// RequestLogger returns a chi middleware that logs requests with zerolog.
func RequestLogger(log zerolog.Logger) func(http.Handler) http.Handler {
	return func(next http.Handler) http.Handler {

				Str("method", r.Method).
				Str("path", r.URL.Path).
				Int("status", ww.Status()).
				Int("bytes", ww.BytesWritten()).
				Dur("dur", time.Since(start)).
				Msg("http")
		})
	}
}
```

```go
package httpserver

import "net/http"

const headerAPIKey = "X-API-Key"

// authMiddleware rejects requests without a matching X-API-Key header.
// The key itself is read from deps.APIKey, which comes from the
// WACLAW_API_KEY environment variable. The old waclaw Node uses the
// same header and same env var, so the frontend proxy and daemon TS
// work unchanged.
func (s *Server) authMiddleware(next http.Handler) http.Handler {
	return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		got := r.Header.Get(headerAPIKey)
		if got == "" || got != s.deps.APIKey {
			writeError(w, http.StatusUnauthorized, "Invalid API key")
			return
		}
		next.ServeHTTP(w, r)
	})
}
```

```go
package httpserver

import (
```

```go
// handleAuth: POST /sessions/:id/auth — starts the pairing flow.
// Calls Session.Connect and waits up to 20s for either a QR code or a
// connected state, then returns {qr, state} or {state: "connected"}.
func (s *Server) handleAuth(w http.ResponseWriter, r *http.Request) {
	id := chi.URLParam(r, "id")
	sess, err := s.deps.Manager.Get(id)
	if err != nil {
		writeError(w, http.StatusNotFound, err.Error())
		return
	}

	// Kick off connect in background — we need the goroutine because
	// Connect may block briefly on network.
	go func() {
		if err := sess.Connect(r.Context()); err != nil {
			s.deps.Log.Warn().Err(err).Str("session", id[:8]).Msg("connect")
		}
	}()

	// Wait up to 20s for QR to appear or already-connected transition.
	deadline := time.Now().Add(20 * time.Second)
	tick := time.NewTicker(100 * time.Millisecond)
	defer tick.Stop()
	for time.Now().Before(deadline) {
		state := sess.State()
		if state == session.StateConnected {
			writeJSON(w, http.StatusOK, map[string]any{"state": "connected"})
			return
		}
		if qr := sess.LastQR(); qr != "" {
			writeJSON(w, http.StatusOK, map[string]any{"qr": qr, "state": "waiting_qr"})
			return
		}
		<-tick.C
	}

	writeJSON(w, http.StatusOK, map[string]any{"state": sess.State(), "error": "timeout waiting for QR or connect"})
}

// handleQR: GET /sessions/:id/qr — poll current QR/state.
func (s *Server) handleQR(w http.ResponseWriter, r *http.Request) {
	id := chi.URLParam(r, "id")
	sess, err := s.deps.Manager.Get(id)
	if err != nil {
		writeError(w, http.StatusNotFound, err.Error())
		return
	}
	out := map[string]any{
		"state": sess.State(),
	}
	if qr := sess.LastQR(); qr != "" {
		out["qr"] = qr
	}
	writeJSON(w, http.StatusOK, out)
}
```

```go
// handleMedia: GET /sessions/:id/media/:jid/:msgId.
// Streams the locally cached media file. If not yet downloaded, enqueues
// a job and returns 425 Too Early so the caller can retry.
func (s *Server) handleMedia(w http.ResponseWriter, r *http.Request) {
	id := chi.URLParam(r, "id")
	jid := chi.URLParam(r, "jid")
	msgID := chi.URLParam(r, "msgId")
	if decoded, err := url.QueryUnescape(jid); err == nil {
		jid = decoded
	}
	if decoded, err := url.QueryUnescape(msgID); err == nil {
		msgID = decoded
	}

	sess, err := s.deps.Manager.Get(id)
	if err != nil {
		writeError(w, http.StatusNotFound, err.Error())
		return
	}

	m, err := sess.Store().GetMessageByID(jid, msgID)
	if err != nil || m == nil {
		writeError(w, http.StatusNotFound, "Message not found")
		return
	}

	if m.LocalPath == "" {
		// Not yet downloaded — the MediaQueueRunner will pick it up on
		// next delivery but we can also kick it explicitly. The daemon
		// TS retries on 425 so this is responsive enough.
		writeError(w, http.StatusTooEarly, "Media not yet downloaded; retry shortly")
		return
	}
	if _, err := os.Stat(m.LocalPath); err != nil {
		writeError(w, http.StatusNotFound, "File missing on disk")
		return
	}

	mime := m.MimeType
	if idx := strings.Index(mime, ";"); idx > 0 {
		mime = strings.TrimSpace(mime[:idx])
	}
	if mime == "" {
		mime = guessMime(m.LocalPath)
	}

	w.Header().Set("Content-Type", mime)
	w.Header().Set("Cache-Control", "private, max-age=3600")
	http.ServeFile(w, r, m.LocalPath)
}

// guessMime provides a fallback MIME type based on file extension.
// Matches waclaw Node's guessMime() in src/index.js for parity.
func guessMime(path string) string {
	switch strings.ToLower(filepath.Ext(path)) {
	case ".oga", ".ogg":
		return "audio/ogg"
	case ".mp3":
		return "audio/mpeg"
	case ".m4a":
		return "audio/mp4"
	case ".jpg", ".jpeg":
		return "image/jpeg"
	case ".png":
		return "image/png"
	case ".webp":
		return "image/webp"
	case ".mp4":
		return "video/mp4"
	case ".webm":
		return "video/webm"
	case ".pdf":
		return "application/pdf"
	default:
		return "application/octet-stream"
	}
}
```

```go
package httpserver

import (
	"fmt"
	"net/http"
	"time"
)

// handleEventsSSE: GET /events
//
// SSE endpoint that broadcasts every event published on the bus to all
// connected subscribers. Each event is serialized as a single SSE
// "data:" line followed by a blank line (RFC-compliant framing):
//
//     data: {"type":"message","session_id":"...","message":{...}}\n
//     \n
//
// Authentication is inline: we accept X-API-Key header OR ?token=KEY
// query param because EventSource in browsers cannot set custom headers.
// The daemon TS in packages/daemon uses fetch() and sets the header,
// so both paths are supported.
//
// Heartbeats every 15s keep intermediate proxies from closing the conn.
func (s *Server) handleEventsSSE(w http.ResponseWriter, r *http.Request) {
	// Inline auth: header or query param.
	key := r.Header.Get(headerAPIKey)
	if key == "" {
		key = r.URL.Query().Get("token")
	}
	if key != s.deps.APIKey {
		writeError(w, http.StatusUnauthorized, "Invalid API key")
		return
	}

	flusher, ok := w.(http.Flusher)
	if !ok {
		writeError(w, http.StatusInternalServerError, "streaming unsupported")
		return
	}

	w.Header().Set("Content-Type", "text/event-stream")
	w.Header().Set("Cache-Control", "no-cache")
	w.Header().Set("Connection", "keep-alive")
	w.Header().Set("X-Accel-Buffering", "no") // disable nginx proxy buffering
	w.WriteHeader(http.StatusOK)

	// Subscribe with a buffered channel so the bus does not drop events
	// while we write to the (potentially slower) network.
	ch, unsub := s.deps.Bus.Subscribe(64)
	defer unsub()

	// Initial comment — some proxies need at least one byte to flush
	// headers.
	fmt.Fprintf(w, ": subscribed\n\n")
	flusher.Flush()

	heartbeat := time.NewTicker(15 * time.Second)
	defer heartbeat.Stop()

	for {
		select {
		case <-r.Context().Done():
			return
		case evt, ok := <-ch:
			if !ok {
				return
			}
			fmt.Fprintf(w, "data: %s\n\n", evt.Raw)
			flusher.Flush()
		case <-heartbeat.C:
			fmt.Fprintf(w, ": ping\n\n")
			flusher.Flush()
		}
	}
}
```

```go
// Package scheduler polls Supabase for pending scheduled messages and
// dispatches them through the session manager. Port of the waclaw Node
// src/scheduler.js, same table (`waclaw_scheduled_messages`), same
// claim-then-send-then-update flow.
package scheduler

import (
	"bytes"
	"context"
	"encoding/json"
	"fmt"
	"io"
	"net/http"
	"time"

	"github.com/andrefogelman/waclaw-go/internal/session"
	"github.com/rs/zerolog"
)

const (
	pollInterval = 30 * time.Second
	batchSize    = 10
)

// Scheduler runs a polling loop against Supabase's REST API. Uses the
// service role key, so it bypasses RLS. One instance per process.
type Scheduler struct {
	mgr         *session.Manager
	supabaseURL string
	serviceKey  string
	log         zerolog.Logger
}

// New returns a scheduler or nil if Supabase env vars are missing
// (scheduler disabled). Callers must check for nil before calling Run.
func New(mgr *session.Manager, supabaseURL, serviceKey string, log zerolog.Logger) *Scheduler {
	if supabaseURL == "" || serviceKey == "" {
		log.Warn().Msg("scheduler disabled: SUPABASE_URL or SUPABASE_SERVICE_ROLE_KEY not set")
		return nil
	}
	return &Scheduler{
		mgr:         mgr,
		supabaseURL: supabaseURL,
		serviceKey:  serviceKey,
		log:         log.With().Str("component", "scheduler").Logger(),
	}
}

// Run polls every pollInterval until ctx is cancelled.
func (s *Scheduler) Run(ctx context.Context) {
	s.log.Info().Msg("scheduler starting")
	ticker := time.NewTicker(pollInterval)
	defer ticker.Stop()

	// First tick after 5s so startup is quick if there are pending msgs.
	timer := time.NewTimer(5 * time.Second)
	defer timer.Stop()

	for {
		select {
		case <-ctx.Done():
			return
		case <-timer.C:
			s.tick(ctx)
		case <-ticker.C:
			s.tick(ctx)
		}
	}
}

type scheduledMsg struct {
	ID               string `json:"id"`
	WaclawSessionID  string `json:"waclaw_session_id"`
	ChatJID          string `json:"chat_jid"`
	Text             string `json:"text"`
	ScheduledFor     string `json:"scheduled_for"`
}

func (s *Scheduler) tick(ctx context.Context) {
	nowISO := time.Now().UTC().Format(time.RFC3339)
	url := fmt.Sprintf("%s/rest/v1/waclaw_scheduled_messages?status=eq.pending&scheduled_for=lte.%s&order=scheduled_for.asc&limit=%d",
		s.supabaseURL, nowISO, batchSize)

	req, _ := http.NewRequestWithContext(ctx, http.MethodGet, url, nil)
	req.Header.Set("apikey", s.serviceKey)
	req.Header.Set("Authorization", "Bearer "+s.serviceKey)

	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		s.log.Warn().Err(err).Msg("poll query")
		return
	}
	defer resp.Body.Close()
	if resp.StatusCode != 200 {
		body, _ := io.ReadAll(resp.Body)
		s.log.Warn().Int("status", resp.StatusCode).Str("body", string(body)).Msg("poll non-200")
		return
	}

	var due []scheduledMsg
	if err := json.NewDecoder(resp.Body).Decode(&due); err != nil {
		s.log.Warn().Err(err).Msg("poll decode")
		return
	}
	if len(due) == 0 {
		return
	}
	s.log.Info().Int("count", len(due)).Msg("processing due")

	for _, msg := range due {
		s.process(ctx, msg)
	}
}

func (s *Scheduler) process(ctx context.Context, msg scheduledMsg) {
	// Claim: update status='processing' where status='pending' AND id=...
	if !s.updateStatus(ctx, msg.ID, "processing", "", true) {
		return
	}

	sess, err := s.mgr.Get(msg.WaclawSessionID)
	if err != nil {
		s.log.Warn().Err(err).Str("id", msg.ID).Msg("session not found")
		s.updateStatus(ctx, msg.ID, "failed", "session not found", false)
		return
	}
	if _, err := sess.SendText(ctx, msg.ChatJID, msg.Text); err != nil {
		s.updateStatus(ctx, msg.ID, "failed", err.Error(), false)
		return
	}
	s.updateStatus(ctx, msg.ID, "sent", "", false)
}

// updateStatus PATCHes the row. claim=true adds a status=pending filter
// so we only claim rows still in pending state (prevents double-sends
// if two schedulers race).
func (s *Scheduler) updateStatus(ctx context.Context, id, status, errMsg string, claim bool) bool {
	body := map[string]any{"status": status}
	if status == "sent" {
		body["sent_at"] = time.Now().UTC().Format(time.RFC3339)
	}
	if errMsg != "" {
		if len(errMsg) > 500 {
			errMsg = errMsg[:500]
		}
		body["error"] = errMsg
	}
	bodyBytes, _ := json.Marshal(body)

	url := fmt.Sprintf("%s/rest/v1/waclaw_scheduled_messages?id=eq.%s", s.supabaseURL, id)
	if claim {
		url += "&status=eq.pending"
	}
	req, _ := http.NewRequestWithContext(ctx, http.MethodPatch, url, bytes.NewReader(bodyBytes))
	req.Header.Set("apikey", s.serviceKey)
	req.Header.Set("Authorization", "Bearer "+s.serviceKey)
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Prefer", "return=minimal")

	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		s.log.Warn().Err(err).Msg("update")
		return false
	}
	defer resp.Body.Close()
	if resp.StatusCode >= 400 {
		body, _ := io.ReadAll(resp.Body)
		s.log.Warn().Int("status", resp.StatusCode).Str("body", string(body)).Msg("update non-2xx")
		return false
	}
	return true
}
```
